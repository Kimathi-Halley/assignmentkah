# 2 Explain the meaning of the following terms

(a) Propositional Variables These are symbols p, q, or r that stand for statements which can be true or false. (b) Truth Table
A table showing all possible truth values of one or more propositions and how they affect any proposition.
(c) Compound Propositions
New statements formed by combining simpler propositions using and, “or, not, their negations etc.
(d) Negation Operator
An operator (usually “¬”) that flips a proposition’s truth value: true becomes false, and false becomes true.
(e) Connectives
Logical words or symbols (e.g., “¬”, “∧”, “∨”, “→”) used to join or modify propositions to form compound statements.

#6 Show (either using Truth Tables or otherwise) that the following pairs are logically equivalent:

(i) ∀x ¬Q(x) and ¬∃x Q(x)

Statement: “For all x, Q(x) is false” is logically equivalent to “There is no x for which Q(x) is true.”
Formally, ¬∃x Q(x) ⇔ ∀x ¬Q(x).

This follows from the standard rule of negating quantifiers: ¬∃x Q(x) is read as “It is not the case that there exists x such that Q(x),” which is exactly “For every x, Q(x) is false.”

(ii) p → q and ¬p ∨ q

Statement: “If p then q” is logically equivalent to “Not p or q.”
We can see this by the following truth table:

p q p → q ¬p ¬p ∨ q
T T T F T
T F F F F
F T T T T
F F T T T

Since the “p → q” column matches exactly with the “¬p ∨ q” column, p → q ≡ ¬p ∨ q.

(iii) p ↔ q and ¬p → ¬q

Clarification: Typically, p ↔ q (read “p if and only if q”) is equivalent to “(p → q) and (q → p).” Each implication can be written in its contrapositive form:

  • p → q is equivalent to ¬q → ¬p
  • q → p is equivalent to ¬p → ¬q
Hence: p ↔ q ≡ (p → q) ∧ (q → p) is also ≡ (¬q → ¬p) ∧ (¬p → ¬q).

If the question states “Show that p ↔ q and ¬p → ¬q are equivalent,” remember you also need the other direction (¬q → ¬p) to capture “if and only if.” In other words: p ↔ q alone is not the same as ¬p → ¬q alone, but: p ↔ q ≡ (¬p → ¬q) ∧ (¬q → ¬p).


#7

(a) Explanation of Quantifiers:

  1. Universal Quantifier: ∀ (for all). For example, ∀x P(x) means "P(x) holds for every x."
  2. Existential Quantifier: ∃ (there exists). For example, ∃x P(x) means "There is at least one x for which P(x) holds."
  3. Uniqueness Quantifier: ∃! (there exists exactly one). For example, ∃!x P(x) means "There exists exactly one x such that P(x) holds."

(b) Let P(x) denote "x + 1 = x" with the domain of all real numbers.

For any real number x, the equation x + 1 = x implies 1 = 0, which is a contradiction. Hence, P(x) is false for every x ∈ ℝ, and the universal statement ∀x ∈ ℝ, (x + 1 = x) is False.

(c) Let P(x) denote "x² = 0" with the domain of all integers.

Consider x = 1 (an integer): 1² = 1, which is not equal to 0. Thus, P(1) is false. Since not every integer satisfies x² = 0, the universal statement ∀x ∈ ℤ, (x² = 0) is False.

Question: If Q(x) denotes the statement “x = x + 1,” what is the truth value of ∃x Q(x) (there exists an x such that x = x + 1) where the domain is all real numbers?

Answer: The statement “x = x + 1” would imply “0 = 1,” which is impossible over the real numbers. Hence, there is no real x satisfying x = x + 1. Therefore, ∃x Q(x) is False

#8 (a) Translate the statement, “Every real number except zero has a Multiplicative
Inverse” (a Multiplicative Inverse of a real number x is a real number y such that
xy = 1).

(a) Translation:
The statement “Every real number except zero has a Multiplicative Inverse” can be written in logical form as:

∀x ∈ ℝ, [x ≠ 0 → ∃y ∈ ℝ (x·y = 1)]

“For every real number x, if x is not zero, then there exists some real number y such that the product of x and y is 1.”

(b) Logical Expression:
∀x [(F(x) ∧ P(x)) → ∃y M(x, y)]

Where:
• F(x): “x is female”
• P(x): “x is a parent”
• M(x, y): “x is the mother of y”

(c) Determine the Truth Value of each statement (Domain: all integers):

  1. ∀n (n+1 > n)
    This is always true for any integer n. Adding 1 to n always yields a strictly greater integer.
  2. ∃n (2n = 3n)
    This is true because n = 0 satisfies 2n = 3n (both sides are 0).
  3. ∃n (n = –n)
    This is also true because n = 0 satisfies n = –n.
  4. ∀n (n² ≥ n)
    This is true for all integers. For nonnegative n, n² ≥ n clearly holds. For negative n, n² is positive (or zero), which is always ≥ n when n is negative or zero.
